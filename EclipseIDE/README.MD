# Eclipse

## Installation
- Download Eclipse Java EE
- <img src="./forReadme/img1.png" />
- Unarchive using `tar -xvf ./eclipse-jee-2023-12-R-linux-gtk-x86_64.tar.gz `
- `cd eclipse`
- `chmod +x ./eclipse`
- Run using: `./eclipse`
